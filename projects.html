Anti-refugee violence in Germany dashboard

Ongoing project

This dashboard visualises each known instance of anti-refugee violence in German since 2014, both across space and over time. To create the app, clone the project, install all dependencies, and run python -m app. To download the most recent attacks, run python -m arvig.

The impact of refugee accommodation on local neighbourhood appeal

Ongoing project

The opening of refugee shelters is regularly met with protest from the surrounding community, in part because residents fear that the presence of nearby shelters may devalue the local neighbourhood. In this projeect I combine precise information on the place and timing of new property listings on a large real estate website with information on the opening of refugee accommodation sites to assess whether the opening of a new shelter affects the desirability of the surrounding area.

Ukraine Crisis: Monitoring population displacement through social media activity

Ongoing project (w. Doug Leasure et al)

The aim of this project is to produce daily sub-national population estimates for Ukraine disaggregated by age and sex based on social media user activity. The report describes the methodology developed for this purpose and the supplementary materials include daily population estimates for 5-year age-sex groups in each Oblast, figures highlighting daily demographic changes, and GIS files for mapping these results.
